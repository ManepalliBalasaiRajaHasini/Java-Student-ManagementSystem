# Student Management System

## Overview

Student Management System is a Java-based desktop application developed using Java Swing. This project helps manage student records through a simple graphical user interface.

## Features

* Add Student Records
* View All Students
* Search Student by ID
* Delete Student Records
* Save Student Data to File
* Load Saved Data Automatically
* User-Friendly GUI using Java Swing

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)
* ArrayList
* File Handling
* Serialization

## Project Structure

```text
StudentManagementSystem
│
├── Student.java
├── StudentManager.java
└── Main.java
```

## How to Run

1. Compile all Java files:

```bash
javac Student.java
javac StudentManager.java
javac Main.java
```

2. Run the application:

```bash
java Main
```
